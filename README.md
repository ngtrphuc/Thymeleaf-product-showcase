# Thymeleaf Product Showcase

## 📌 Overview

This is a simple product showcase website built with Spring Boot and Thymeleaf.

## 🚀 Features

* Product list page
* Product detail page
* Clean MVC structure

## 🛠️ Tech Stack

* Java
* Spring Boot
* Thymeleaf
* Gradle

## ⚙️ How to Run

```bash
./gradlew bootRun
```

Then open:
http://localhost:8080
